# Motor Driver Controller PCB

A custom MOSFET-based motor driver board designed for the Northeastern Rover Club, providing bidirectional DC motor control for rover drivetrain and actuator systems.

## Overview

This board was designed to allow the cs team t acruently control the motors that integrates directly into the rover's main electrical system. It handles motor driving, voltage regulation, and provides onboard debugging/testing interfaces.

## Key Features
- Bidirectional motor control using MOSFETs and BJTs
- Onboard voltage regulation — 15 V → 5 V and 5 V → 3.3 V
- USB-C port for communication
- SWD debug header for firmware flashing and automated design verification
- Compact layout thats optimized for integration into the rover's main electrical board

## Key Skills Gained
- Reading mechanical drawings to create custom schematic components
- Simulation testing for circuit validation before fabrication, in Altium Designer
- PCB layout and routing in DipTrace 
- PCB assembly, soldering, and board bring-up
